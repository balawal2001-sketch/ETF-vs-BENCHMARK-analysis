# ETF-vs-BENCHMARK-analysis by using Jupiter notebook (PYTHON)
ETF Performance Analysis – CSPX

Analisi della performance dell'ETF CSPX.L (iShares S&P 500 UCITS ETF) negli ultimi 5 anni, con confronto rispetto all'indice S&P 500 Total Return (^SP500TR) e valutazione statistica dei rendimenti.

Obiettivo

Valutare:

1.Rendimenti cumulati semplici
2.Volatilità giornaliera
3.Tracking difference rispetto all'indice
4.Beta e normalità dei rendimenti
